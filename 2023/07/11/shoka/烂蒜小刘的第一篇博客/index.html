修改自定义背景

# 前言

原主题的背景默认是设置的渐变纯色，鉴于个人喜好，将背景换成自定义图片

修改自定义背景，实际上就是将各组件部分的背景和边框样式删除，并用一个背景层（设置背景图片）和一个遮罩层（设置透明度）来实现
可根据自己的喜好删除相应的背景和边框

# 实现

# 修改 nunjucks (layout)

  1. 找到 themes\shoka\layout\_partials\ 目录；

  2. 修改 layout.njk ，在 main 和 footer 部分外面添加 <div id="bgpic"> 和 <div id='mask'>

    直接使用时，记得补全省略号中的原代码

# 修改 Stylus

  1. 找到 themes/shoka/source/css/_common/outline 目录；
  2. 修改 outline.styl ，在合适位置添加 #bgpic 和 #mask （添加样式和背景图片）
  3. 修改 main 部分，将下面语句注释（设置 main 部分背景透明）
  4. 修改 #mian 部分，将下面语句注释（设置文章主题部分及边框背景透明）
  5. 找到 themes/shoka/source/css/_common/outline/footer 目录
  6. 修改 footer.styl 中 #footer 部分，将下面语句注释（设置 footer 部分背景透明）
  7. 找到 themes/shoka/source/css/_common/components/post 目录
